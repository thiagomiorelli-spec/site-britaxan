Site Britaxan

Site institucional desenvolvido para a empresa Britaxan, onde trabalho atualmente, especializada em britas, concreto e argamassa.

Sobre o projeto

Site desenvolvido para ser claro e objetivo, pensado para que o público-alvo da empresa: construtores, empreiteiros e clientes de obras, encontre as informações que precisam de forma rápida e direta.
O site ainda não está no ar.
A publicação depende de decisão da empresa sobre hospedagem e domínio.

Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

Funcionalidades
- Menu de navegação fixo com âncoras
- Slideshow de imagens
- Seção de produtos com cards
- Seção Nossa História com vídeo do YouTube
- Botões integrados com WhatsApp
- Rodapé com endereços e redes sociais

Layout
- Identidade visual baseada na logo da empresa
- Cores: preto, vermelho e cinza
- Fonte: Barlow Condensed (Google Fonts)
